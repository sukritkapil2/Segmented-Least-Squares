Algorithm

 

Given a set of points P = {(x1, y1), (x2, y2), , (xn, yn)}, where x1 ≤ x2 ≤ ≤ xn, we partition the point set into some number of segments. Each segment is a subset of P that represents a contiguous set of x-coordinates i.e., a subset of the form, {(xi, yi), (xi+1, yi+1), , (xj, yj)} for some i ≤ j.

For each segment S, we find the line minimizing the squared error w.r.t the points in S with the formula as stated above

The penalty of a partition is defined to be the sum of the following two terms:

1.     The number of segments multiplied by a given multiplier C > 0.

2.     For each segment, the error value of the optimal line through that segment.

The goal is to find a partition of the minimum penalty. While increasing the number of segments reduces the penalty term (2), it increases the penalty term (1). The multiplier C is part of the input.

The main function that calculates the least square is FindSegmentedLeastSquares()
